# Transformer Dashboard

## 🚀 Setup

```bash
npm install
```

## ▶️ Run locally

```bash
npm run dev
```

## 🧱 Build

```bash
npm run build
```

## 🔍 Preview production build

```bash
npm run preview
```

## 🐳 Docker

Build and run container:

```bash
docker build -t transformer-dashboard .
docker run -p 4173:4173 transformer-dashboard
```
